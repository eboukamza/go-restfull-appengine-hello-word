 - GO (with mux)
 - Appengine + DataStore

API :
----

 - `[GET] /api/hellos`

 - `[GET] /api/hellos/{id}`

 - `[POST] /api/hellos
   {
    "name":"your_name"
   }`

 - `[PUT] /api/hellos/{id}`

 - `[DELETE] /api/hellos/{id}`
